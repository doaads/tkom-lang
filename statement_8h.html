Detailed Description

Contains the definition of the Statement class and related structures.

This file defines the Statement class, which serves as the base class for various types of statements in the abstract syntax tree (AST). and type aliases used for working with statements. Each type of statement (e.g., assignment, conditionals, loops) is represented by a subclass of Statement and implements the accept method for interacting with a Visitor.

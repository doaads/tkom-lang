\doxysubsection{Detailed Description}
Represents a unary expression (e.\+g., negation or logical NOT). 

The {\ttfamily \doxylink{classUnaryExpr}{Unary\+Expr}} class is used to represent unary operations, such as negation ({\ttfamily !}) or logical NOT. It stores the operator type and the right-\/hand operand (which is a sub-\/expression).

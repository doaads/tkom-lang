\doxysubsection{Detailed Description}
Contains the definition of the \doxylink{classStatement}{Statement} class and related structures. 

This file defines the {\ttfamily \doxylink{classStatement}{Statement}} class, which serves as the base class for various types of statements in the abstract syntax tree (AST). and type aliases used for working with statements. Each type of statement (e.\+g., assignment, conditionals, loops) is represented by a subclass of {\ttfamily \doxylink{classStatement}{Statement}} and implements the {\ttfamily accept} method for interacting with a {\ttfamily \doxylink{classVisitor}{Visitor}}.

\doxysubsection{Detailed Description}
Represents a function call expression. 

The {\ttfamily \doxylink{classCallExpr}{Call\+Expr}} class is used to represent function calls in the source code. It stores the function name (which is an expression) and the list of arguments (which are also expressions).

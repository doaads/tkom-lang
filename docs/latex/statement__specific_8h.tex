\doxysubsection{Detailed Description}
Contains specific statement classes representing various control flow and function-\/related constructs. 

This file defines concrete subclasses of {\ttfamily \doxylink{classStatement}{Statement}} that represent specific control flow structures such as for-\/loops, while-\/loops, conditionals, return statements, and function calls.

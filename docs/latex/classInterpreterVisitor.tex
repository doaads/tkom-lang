\doxysubsection{Detailed Description}
Interpreter implementation. 

The interpreter goes through the parsed synatx tree and ensures the following\+:
\begin{DoxyItemize}
\item strong assignment -\/ the variable\textquotesingle{}s type cannot be changed as runtime
\item weak typing -\/ implicit casting
\item passing variables as a reference 
\end{DoxyItemize}
